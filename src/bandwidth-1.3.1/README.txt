Run 'make cachemisses64' to build the binary 'cachemisses64'

cachemisses64 simply allocates 200MB memory and does random reads aligned to 64
bytes boundary (cache line size) for 1 millions times using assembly code
generated by gen-asm.py and saved as cachemisses64.asm.

To gather LLC loads and misses data of cachemisses64, run: 'perf stat -e LLC-loads,LLC-load-misses ./cachemisses64'
